# Italy QR Quest

Italy QR Quest is a mobile-first QR scavenger hunt web app built for live events.

## Features

- 10 location checkpoints
- One mini-game per location
- Progress tracking
- Certificate unlock on full completion
- Shared leaderboard (Firebase)

## Tech Stack

- Vite
- React
- React Router
- CSS
- Firebase
